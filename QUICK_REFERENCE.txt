TESTING QUICK REFERENCE
================================================================================

SETTINGS - PROFILE:
  1. Go to /account/settings
  2. See current profile values displayed
  3. Click "Edit" button
  4. Change a field (e.g., First Name)
  5. Click "Review Changes"
  6. See confirmation showing old→new values
  7. Click "Confirm Changes"
  8. Verify changes saved and page refreshed

SETTINGS - PASSWORD:
  1. Go to /account/settings
  2. Click "Change Password" button
  3. Try entering wrong current password
  4. See error: "Current password is incorrect"
  5. Enter correct current password
  6. Click "Verify Password"
  7. See green success message: "✓ Password verified"
  8. Enter new password (must be different from current)
  9. Confirm new password (must match)
  10. Click "Change Password"
  11. See success message
  12. Try signing out and in with new password

CART - CLEAR:
  1. Add 3+ items to cart
  2. Click "Clear Cart" button
  3. See confirmation: "This will remove all X items"
  4. Click "Keep Items"
  5. Verify items still in cart
  6. Click "Clear Cart" again
  7. Click "Clear Cart" in confirmation
  8. Verify cart now empty

CART - REMOVE ITEM:
  1. Add items to cart
  2. Click trash icon on one item
  3. See confirmation with item name
  4. Click "Keep Item"
  5. Verify item still in cart
  6. Click trash icon again
  7. Click "Remove" in confirmation
  8. Verify only that item was removed

ADDRESSES - DELETE:
  1. Go to /account/addresses
  2. Click "Delete" on any address
  3. See confirmation with full address details
  4. Click "Cancel"
  5. Verify address still exists
  6. Click "Delete" again
  7. Click "Delete" in confirmation
  8. Verify address deleted

ADDRESSES - SET DEFAULT:
  1. Go to /account/addresses (with 2+ addresses)
  2. Click "Set Default" on non-default address
  3. See confirmation showing which address
  4. Click "Cancel"
  5. Verify address is NOT now default
  6. Click "Set Default" again
  7. Click "Set Default" in confirmation
  8. Verify address now marked as default

WISHLIST:
  1. Add item to wishlist
  2. Go to /account/wishlist
  3. Click "Remove"
  4. See confirmation
  5. Click "Keep Item"
  6. Verify item still there
  7. Click "Remove" again
  8. Click "Remove" in confirmation
  9. Verify item removed
