## HOW TO PREPARE FOR SUBMISSION

### Step 1: Open Report File
```
1. Open FINAL_REPORT_FORMATTED.md in MS Word
   (Right-click → Open with → Microsoft Word)
2. Or copy-paste entire content into new Word document
3. Save as: "HTM_ECOMM_Final_Report.docx"
```

### Step 2: Apply Formatting

**Font and Size:**
- Select All: Ctrl+A
- Font: Times New Roman
- Size: 12pt

**Margins:**
- Layout → Margins → Custom Margins
- Left: 3.5 cm
- Top: 2.5 cm
- Right: 1.25 cm
- Bottom: 1.25 cm

**Spacing:**
- Home → Line Spacing → 2.0

### Step 3: Print Report (91 pages)

**Print Settings:**
- Paper Size: A4 (210 × 297 mm)
- Orientation: Portrait
- Print Quality: High
- Print one-sided only
- Use paper: 80 gsm minimum

**Printing Command:**
```
File → Print
- Printer: Select laser/inkjet printer
- Paper: A4
- Margins: As set above
- Print Range: All pages
- Copies: 1 only
```

### Step 4: Print Cover Page (Separate)

**Cover Page Details:**
- Print on heavy cardstock (280 gsm preferred)
- All text in black ink
- Fill in student names, roll numbers, guide name, dates
- Use as outermost layer when binding

### Step 5: Bind Report

**Spiral Binding:**
1. Arrange all 91 pages in correct order
2. Place cover page as first (outermost) page
3. Align all pages perfectly
4. Use spiral binding machine (professional or commercial)
5. Bind on left side with 1-1.5 inch coil

**Result:** Single professional-looking bound report ready for submission
